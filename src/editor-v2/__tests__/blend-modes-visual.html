Blend-mode visual harness

Renders all 16 ids from BlendModes.BLEND_MODES against a fixed base + blend pair. Native-GPU modes render through Pixi's default path; the four HSL modes fall back to normal with a console.info until their filter wiring ships in a post-Phase 1 polish pass (tracked in the Phase 1 queue).

This file is a manual harness — open in a browser to inspect. CI verification is via Phase 1's Jest suites which cover the data-model and resolver contract.
